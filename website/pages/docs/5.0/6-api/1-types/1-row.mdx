#### Definition

```ts
type Row = {
  rowIndex: number;
  height: string | number;
  reorderable?: boolean;
};
```
#### Properties

| Name          | Type                 | Description                            |
| ------------- | -------------------- | -------------------------------------- |
| `rowIndex`    | `number`             | The index of the row.                  |
| `height`      | `string \| number`   | The height of the row.                 |
| `reorderable` | `boolean (optional)` | Indicates if the row can be reordered. |
